# Version Control — Always On

## FILE BACKUP RULE — ALWAYS ON

Before editing ANY existing file:
- Back it up first
- Save a copy with the filename plus the date and time appended
- Example: `command-center-v81-backup-20260409.jsx`
- Example: `CLAUDE-backup-20260409.md`

Never overwrite a working file without a backup. No exceptions.

If something breaks after an edit:
- Restore the backup immediately without being asked
- Do not attempt further edits until the working state is restored
- Then diagnose the problem on the backup copy before re-attempting the edit

Backup location:
- Same directory as the original file unless Renée specifies otherwise
